# 指標解説 — Macro Cockpit の計器の読み方

このドキュメントは、ダッシュボードの25パネルとレジーム・ストリップを毎朝どう読むかをまとめた計器説明書である。目的は個々の指標の教科書的解説ではなく、「運用の前提シナリオが崩れていないか」を判断するための読み方に絞る。各指標は「これは何か/何がわかるか/シナリオとの関係/読み方の注意」の順に記す。ソース・頻度・計算方法は実装(`fetcher/src/fetcher/registry.py`、`web/lib/panels.ts`、`config/regime.yaml`)と一致させてあり、食い違いがあれば実装側が正である。

監視する前提シナリオは SPEC §1 の3つである。

- **スタグフレーション** — 成長が鈍いままインフレが粘着・再加速する世界。米CPI YoY・BEI・実質金利(DFII10)・金・コモディティ群で監視する。インフレ再加速×実質金利低下×金高が継続の典型形。
- **財政ドミナンス** — 財政の持続性懸念が金融政策を縛り、長期金利の上昇圧力とB/S再拡大が併存する世界。JGB10年/30年、日米金利差、Fed・日銀B/S、米10年金利、HYスプレッドで監視する。特にJGB30年は懸念のセンサーである。
- **円希薄化** — 円の購買力が構造的に薄まる世界。USD/JPY、日CPI、マネタリーベース、実質賃金、貿易収支、金(円建て資産の代替)で監視する。円安×日CPI上昇×実質賃金マイナスの併存が典型形。

## レジーム・ストリップの読み方

画面最上部の横帯が本画面の核心で、これだけ見れば毎朝の確認が終わる設計である。1セル=1コア指標で、既定は8セル(米CPI/日CPI/JGB10Y/USDJPY/金/Fed B/S/日銀B/S/VIX)。判定ルールはコードでなく `config/regime.yaml` に記述され、編集して保存しブラウザをリロードするだけで反映される。

**色の意味**: 整合=緑(シナリオと同じ方向に動いている)/中立=グレー(変化が僅少、またはデータ欠損)/逆行=赤(シナリオと逆方向、または警戒域)。セル内の▲▼は「実際の方向」を示し、色は整合/逆行の判定結果で決まる。▼が緑になることも(expected: down のセルなら)あり得る点に注意。

**direction 型の既定ルール**: 直近値を「3ヶ月前以前の最も近い観測」(`defaults.lookback_months: 3`)と比較し、変化の方向が期待方向と一致すれば整合。変化幅が基準値の0.1%未満(`flat_epsilon_rel: 0.001`。ゼロ近傍は0.001pt)なら中立。既定の期待方向とその理由は次の通り。

| セル | 期待方向 | 理由 |
|---|---|---|
| 米CPI(YoY) | up | スタグフレーション: インフレ再加速が整合 |
| 日CPI(YoY) | up | スタグフレーション/円希薄化 |
| JGB10Y | up | 財政ドミナンス: 長期金利上昇が整合 |
| USDJPY | up | 円希薄化: 円安が整合 |
| 金 | up | 実物資産インフレが整合 |
| Fed B/S | up | 緩和再拡大が整合 |
| 日銀B/S | up | 緩和再拡大が整合 |
| VIX | threshold(warn_above: 30) | リスクイベント検知。方向は問わず、30超で逆行(赤) |

**サマリの解釈**: 整合セルが `min_aligned: 6` 以上なら「シナリオ継続」、未満なら「要注意」。逆行セルがあれば件数が添えられる。「要注意」=シナリオ破綻ではなく、パネル本文まで降りて原因のセルを特定せよという合図である。なお日CPI・JGB10Yのセルは公式ソース優先のチェーン(e-Stat月次/財務省日次を優先、FREDへフォールバック)で判定される。YAMLが壊れた場合は全セルグレー+「ルール設定エラー」の退避表示になり、ページ自体は落ちない。

## インフレ・金利(6パネル)

### 米CPI YoY

**これは何か**: FRED `CPIAUCSL`(米消費者物価指数・月次)の前年同月比。DBには指数のまま保存し、YoYは表示層で計算する派生値。**何がわかるか**: 米インフレの再加速か鈍化か。Fedの利下げ余地と実質金利の行方を規定する最上流の計器である。**シナリオとの関係**: 上昇はスタグフレーション継続を示唆(レジーム・ストリップ対象、期待方向up)。明確な鈍化トレンド入りはシナリオの前提を疑う材料。**注意**: 月次発表で約2週間のラグがあり、日々動く指標ではない。騰落はpt差で表示される。

### 日CPI YoY

**これは何か**: e-Stat API の月次CPI(2020年基準・全国総合、指数)からYoYを表示層で計算。e-Stat未取得時はFRED `FPCPITOTLZGJPN`(世界銀行・年次・値が既に前年比%)へフォールバックし、その場合「年次」注記と10年チャートに切り替わる。**何がわかるか**: 円の国内購買力の毀損ペース。日銀の政策正常化圧力の強さを測る。**シナリオとの関係**: 上昇はスタグフレーション/円希薄化の継続に整合(ストリップ対象、期待方向up)。**注意**: 前月比表示は無し(YoYのみ)。年次フォールバック中は月次の変化を読めないことを意識する。

### 米10年BEI

**これは何か**: FRED `T10YIE`(日次)。名目10年金利−実質10年金利で定義される市場の期待インフレ率。**何がわかるか**: CPIが実績値であるのに対し、BEIは市場が織り込む将来インフレ。CPIが落ち着いていてもBEIが切り上がるなら、市場はインフレ粘着を疑い始めている。**シナリオとの関係**: 上昇はスタグフレーション継続の先行示唆。BEI低下×名目金利上昇の組み合わせは実質金利上昇を意味し、金にとって逆風のサインとして読む。**注意**: 日次で振れるため、水準変化はトレンドで確認する。

### 米10年金利

**これは何か**: FRED `DGS10`(日次)。米国債10年の名目利回り。**何がわかるか**: 世界の割引率の基準。上昇の中身が「期待インフレ(BEI)」か「実質金利(DFII10)」かで意味が正反対になるため、必ず両パネルと分解して読む。**シナリオとの関係**: 財政要因(タームプレミアム)による上昇は米側の財政ドミナンスの進行を示唆し、日米金利差経由でUSD/JPYにも波及する。**注意**: 単体では判断せず、BEI・実質金利・HYスプレッドとの同時読みが前提の計器である。

### 実質金利

**これは何か**: FRED `DFII10`(日次)。米10年TIPS利回り=名目金利から期待インフレを差し引いた実質金利。**何がわかるか**: 金価格の主ドライバー(逆相関)であり、スタグフレーション判定の中核。実質金利の低下は「インフレに金利が追いつかない」状態を意味する。**シナリオとの関係**: 低下×金高はスタグフレーション/円希薄化の継続に整合。実質金利が明確に上昇トレンドへ転じ金が崩れるなら、シナリオの中核前提を疑う。**注意**: 金パネルとセットで読むこと。騰落はpt差表示。

### 日米金利差

**これは何か**: 表示層で計算する派生値(DBに無い)。DGS10の各観測日について「その日以前で最新のJGB10年」(財務省日次を優先、FRED月次へフォールバック)を差し引いた値。**何がわかるか**: USD/JPYの金利面の推進力。金利差の拡大は円キャリーの妙味増=円安圧力、縮小は円高圧力を示唆する。**シナリオとの関係**: 金利差が縮小してもなお円安が続くなら、金利以外の要因(実需・財政懸念)による円希薄化の進行を疑う強いサインとして読む。**注意**: JGB側が月次フォールバック中は日付整合が粗くなる。

## 通貨・中銀(6パネル)

### USD/JPY

**これは何か**: FRED `DEXJPUS`(日次)。補助系列としてIB(IDEALPROミッド、フォールバックyfinance)の `JPY=X` も収録するが、パネルの正はFRED。**何がわかるか**: 円希薄化シナリオの最重要の総合計器。金利差・実需・財政懸念のすべてが集約される。**シナリオとの関係**: 上昇(円安)は円希薄化の継続に整合(ストリップ対象、期待方向up)。持続的な円高転換はシナリオの根幹を疑う材料。**注意**: 動いた日は必ず日米金利差パネルと突き合わせ、「金利で説明できる円安か」を確認する。

### ドル指数

**これは何か**: FRED `DTWEXBGS`(貿易加重ドル実効指数・日次。SPECは週次と記すが実頻度は日次)。**何がわかるか**: USD/JPYの動きが「ドル全面高」か「円独歩安」かを切り分ける計器。ドル指数が横ばいでUSD/JPYだけ上がるなら円固有の売り圧力である。**シナリオとの関係**: 円独歩安の確認は円希薄化シナリオの純度の高い証拠となる。逆にドル全面高による円安は、シナリオとは別の(ドル側の)要因として割り引いて読む。**注意**: 指数水準そのものより、USD/JPYとの乖離に意味がある。

### Fed B/S

**これは何か**: FRED `WALCL`(Fed総資産・週次)。パネルは兆ドル($T)換算で表示。**何がわかるか**: 米金融環境の量的側面。QTの継続か、停止・再拡大への転換かを示す。**シナリオとの関係**: 再拡大への転換は「インフレ収束前の緩和」=スタグフレーション/財政ドミナンスの継続に整合(ストリップ対象、期待方向up)。QT継続下でのインフレ鈍化はシナリオに逆行する組み合わせ。**注意**: 週次更新のため日々は動かない。短期の技術的要因(MBS決済タイミング、貸出ファシリティ利用等)による振れをトレンドと混同しない。なおTGAは負債側の項目であり、その増減は準備預金との構成の入れ替えにすぎず総資産(WALCL)を直接は動かさない。

### 日銀 B/S

**これは何か**: FRED `JPNASSETS`(日銀総資産・月次、億円)。パネルは兆円換算で表示。**何がわかるか**: 日銀の量的な政策スタンス。縮小が進むか、再拡大に転じるかは円の希薄化ペースを直接規定する。**シナリオとの関係**: 拡大は円希薄化/財政ドミナンス(国債買支え)の継続に整合(ストリップ対象、期待方向up)。明確な縮小トレンドはシナリオの減速を示唆する。**注意**: マネタリーベース(負債側の一部)とは別物。総資産は貸出・ETF等も含む資産側の総量であり、両者の乖離にも情報がある。

### マネタリーベース

**これは何か**: 日銀 stat-search CSV(`BOJ:MB`・月次・億円)。パネルは兆円換算。日銀が供給する通貨の総量(日銀券+貨幣+日銀当座預金)。**何がわかるか**: 円の「分母」の増減そのもの。日銀B/S(資産側総量)に対し、こちらは通貨供給という負債側の計器である。**シナリオとの関係**: 増加基調は円希薄化の量的な裏付けとして整合。減少に転じてなお円安・日CPI上昇が続くなら、希薄化の主因が量から財政・実需へ移った可能性を疑う。**注意**: 月次で動きは緩慢。YoYの符号転換を節目として読む。

### 無担保コールO/N

**これは何か**: 日銀 stat-search CSV(`BOJ:CALLON`・日次・年%)。無担保コール翌日物の加重平均レート。**何がわかるか**: 日銀の政策金利の「実勢」。公表される誘導目標ではなく市場で実際についた金利であり、利上げ・利下げの実効性がここに現れる。**シナリオとの関係**: 実勢金利の切り上がりは日銀の正常化進行を意味し、それでも円安・長期金利上昇が止まらないなら財政ドミナンス的な様相の強まりを示唆する。**注意**: 政策変更日近辺以外はほぼ動かない。階段状の変化だけを追えばよい。

## 実物資産(4パネル)

### 金

**これは何か**: 金先物(一次ソースIB ContFuture COMEX日足、フォールバックyfinance `GC=F`)。日次・ドル建て。**何がわかるか**: 通貨の希薄化に対する市場のヘッジ需要の総量。主ドライバーは米実質金利(DFII10)との逆相関で、中銀の買いと財政懸念が上乗せ要因になる。**シナリオとの関係**: 上昇は3シナリオすべての継続に整合する中核指標(ストリップ対象、期待方向up)。実質金利上昇下でも金が崩れないなら、財政・通貨不信によるサポートを示唆する。**注意**: 必ず実質金利パネルとの対で読む。逆相関が切れたときこそ情報がある。

### WTI原油

**これは何か**: WTI原油先物(一次ソースIB ContFuture NYMEX、フォールバックyfinance `CL=F`)。日次・ドル建て。**何がわかるか**: インフレの供給サイド要因の最大の変数。原油高はCPIに数ヶ月遅れて波及し、日本には交易条件悪化=円売り実需としても効く。**シナリオとの関係**: 上昇はスタグフレーション(コストプッシュ)と円希薄化(貿易収支悪化経由)の継続に整合。需要崩壊による急落はスタグフレーションではなくデフレ型後退への移行を疑わせる。**注意**: 地政学イベントでの瞬間的な急騰はトレンドと区別する。

### 銅

**これは何か**: 銅先物(一次ソースIB ContFuture COMEX、フォールバックyfinance `HG=F`)。日次・ドル建て。**何がわかるか**: 実需に最も敏感なコモディティで、世界の製造業サイクルの体温計。「Dr. Copper」として景気の実勢を映す。**シナリオとの関係**: 銅安×金高の乖離は「景気は弱いのに通貨ヘッジ需要は強い」というスタグフレーションの典型形として読む。銅高を伴う金高なら、リフレ(景気拡大型)の可能性もあり解釈を分ける。**注意**: 単体でなく金との比(金/銅の相対)で局面を判断する計器である。

### 天然ガス

**これは何か**: 米天然ガス先物・Henry Hub(一次ソースIB ContFuture NYMEX、フォールバックyfinance `NG=F`)。日次。アジアLNG(JKM)が有料のため米ガスを代理とする。**何がわかるか**: エネルギーインフレのうち電力・都市ガス経由で家計CPIに効く成分。日本の輸入額(LNG)の先行きの粗い代理でもある。**シナリオとの関係**: 高騰はコストプッシュ型インフレ=スタグフレーションの継続と、貿易収支悪化経由の円希薄化に整合。**注意**: あくまで米国内価格でありJKMとは需給が別。季節性が強く、水準の急変も珍しくないためトレンドで読む。

## 株式・リスク(4パネル)

### 日経225

**これは何か**: yfinance `^N225`(日次)。**何がわかるか**: 円建て資産の名目価格。円安局面では「株高でもドル建て(実質)では目減り」があり得るため、名目の上昇をそのまま健全とは読まない。**シナリオとの関係**: 円安を伴う日経の上昇は円希薄化下の名目資産インフレとして整合的に読む。USD/JPYが横ばいでの下落は国内固有の悪材料、円高転換を伴う急落はシナリオ転換(円希薄化の巻き戻し)の疑いとして扱う。**注意**: S&P500・USD/JPYと同時に読み、ドル建て換算での実力を頭の中で補正する。

### S&P500

**これは何か**: yfinance `^GSPC`(日次)。**何がわかるか**: 世界のリスク資産の基準価格。実質金利との関係(金利上昇に耐えているか)で市場の景気観を測る。**シナリオとの関係**: 実質金利上昇下での株安×金高はスタグフレーション継続に整合。株高×HYスプレッド縮小×VIX低位ならリスク選好は健在で、シナリオの「景気悪化」側はまだ市場に織り込まれていないと読む。**注意**: 指数は少数の大型株に偏っており、指数の堅調が景気全体の健全さを意味しない点は割り引く。

### VIX

**これは何か**: yfinance `^VIX`(日次)。S&P500オプションの予想変動率。**何がわかるか**: 「どこかで騒ぎが起きた」ことの検知器。個別の何が起きたかは示さず、市場全体のストレス水準だけを示す。**シナリオとの関係**: ストリップ対象だが唯一のthreshold型で、方向は問わず30超で逆行(赤)。急騰時はシナリオの前提以前に、ポジションのリスク管理を優先すべき局面である。**注意**: 前日比でなく水準で見る。20未満=平時、30超=警戒。低水準の持続はむしろ油断の蓄積として頭に留める。

### HYスプレッド

**これは何か**: FRED `BAMLH0A0HYM2`(米ハイイールド社債のOASスプレッド・日次・%)。**何がわかるか**: 信用ストレスの先行指標。株より早く「資金繰りの痛み」を織り込むことが多く、VIXが平静でもスプレッドが静かに広がり始めていれば要警戒。**シナリオとの関係**: 拡大は景気悪化側の進行=スタグフレーションの「スタグ」側の顕在化を示唆。金利上昇下でも縮小が続くなら、市場は財政・金利より企業収益を信頼していると読む。**注意**: 水準(%)とともに変化速度を見る。急拡大は水準を問わず警戒に値する。

## 日本・財政(3パネル)

### 日本30年金利

**これは何か**: 財務省 国債金利CSV(`MOF:JGB30Y`・日次)。**何がわかるか**: 財政ドミナンス懸念の最重要センサー(SPEC §4.2)。10年は日銀の政策の影響が強く残るのに対し、30年は財政の持続性への市場の評価がより素直に出る。**シナリオとの関係**: 10年比での急峻化(超長期だけ売られる)は財政ドミナンスの進行の典型的サインとして読む。落ち着いた低下はシナリオの小康を示唆。**注意**: 生保等の需給要因でも動くため、1日の急変は入札・規制イベントの有無を確認してから解釈する。

### 実質賃金 YoY

**これは何か**: 統計ダッシュボードAPI(`ESTATDB:REALWAGE`・毎月勤労統計の実質賃金指数・月次・原数値)の前年同月比。YoYは表示層で計算する派生値。**何がわかるか**: インフレに賃金が追いついているか。マイナス継続は家計の購買力毀損=「インフレはあるが豊かにならない」状態の直接の証拠。**シナリオとの関係**: マイナスの持続はスタグフレーション/円希薄化の継続に整合。明確なプラス定着はシナリオの好転(賃金主導の健全なインフレへの移行)を示唆する。**注意**: 原数値は賞与月に急伸する強い季節性があるためYoYのみ表示(前月比なし)。速報値を含み、後に改定される。

### 貿易収支

**これは何か**: 税関 貿易統計CSV(`CUSTOMS:TRADE`・月次・億円)。輸出総額−輸入総額を取得層で計算して保存。通関ベースであり、国際収支ベースの貿易収支とは定義が異なる。**何がわかるか**: 円の実需フローの構造。恒常的な赤字は「円を売って資源を買う」構造の定着を意味し、金利差に依らない円安圧力の土台になる。**シナリオとの関係**: 赤字の定着・拡大は円希薄化の構造的な裏付けとして整合。黒字転換の持続はシナリオの土台の変化を疑う材料。**注意**: 符号が意味を持つ(パネルは符号付き・0ライン表示)。資源価格と為替の両方で振れるため単月では判断しない。

## 不動産(2パネル)

### 東証REIT

**これは何か**: yfinance `1343.T`(NEXT FUNDS 東証REIT指数連動ETF・日次・円)。**何がわかるか**: 国内不動産の市場ベースの先行プロキシ。実物の不動産統計より数ヶ月早く、金利と賃料見通しへの市場の評価を映す。**シナリオとの関係**: JGB金利上昇下での下落は「金利上昇が実物資産価格を毀損し始めた」ことを示唆し、財政ドミナンスの痛みが国内資産へ及ぶ経路の監視点。金利上昇にもかかわらず堅調なら、インフレ(賃料)期待が金利を上回っていると読む。**注意**: ETF価格であり分配金落ち・需給の影響を含む。JGB10年・30年パネルと対で読む。

### 米住宅CS

**これは何か**: FRED `CSUSHPISA`(S&P Case-Shiller 米住宅価格指数・月次)。**約2ヶ月のラグ**がある。**何がわかるか**: 実物資産インフレの確認用の遅行計器。住宅は家賃経由でCPIの粘着成分(シェルター)を規定するため、住宅価格の再加速はCPI鈍化シナリオの障害になる。**シナリオとの関係**: 高金利下でも上昇が続くなら実物資産インフレの粘着=スタグフレーション継続に整合。明確な下落トレンドはインフレ収束側の証拠。**注意**: ラグが大きく、足元の判断材料ではなく事後確認用。速報性は東証REIT等の市場価格系に譲る。

## 付録: 派生値の計算仕様

派生値はDBに保存せず、すべて表示層で計算する。日付ヘルパーと前年同月比(`computeYoY`)は `web/lib/regime.ts` の共有実装、日米金利差(`computeRateDiff`)と騰落表示(`makeChange`)は `web/lib/panels.ts` にある。

- **前年同月比(米CPI YoY・日CPI YoY・実質賃金 YoY)**: 観測を月キー(YYYY-MM)で索引し、各観測についてちょうど12ヶ月前の同月の値との比 `(当月値 / 前年同月値 − 1) × 100` を%で算出する(`computeYoY`)。前年同月の観測が存在しない月(系列の先頭12ヶ月や欠測月)はプロットされない。日CPIの年次フォールバック(`FPCPITOTLZGJPN`)は値が既に前年比%のため変換しない。
- **日米金利差**: `DGS10`(日次)の各観測日について、「その日以前で最も新しいJGB10年の観測」を差し引く(`computeRateDiff`)。JGB側は財務省日次(`MOF:JGB10Y`)を優先し、無ければFRED月次(`IRLTLT01JPM156N`)へフォールバックするため、フォールバック時は月内の全営業日が同じJGB値と対になる。
- **騰落表示(前月比/前年比)**: 最新観測日から1ヶ月/12ヶ月さかのぼった日付(`shiftMonths`。月末は繰り上げ調整)に対し「その日以前で最も近い観測」(`lastOnOrBefore`)を参照値とする。比率(%)か差分(pt)かはパネル定義の `deltaMode` に従う(金利・スプレッド・YoY系はpt差)。
- **レジーム・ストリップの direction 判定**も同じ日付ヘルパーを用い、直近値と「lookback ヶ月前以前の最も近い観測」の符号付き差分で方向を決める。
